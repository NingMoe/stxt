串列表示式
..........

串列表示式其結果為串列。

:生產子:

生產子格式如下：

x <- xs 

表示由 xs 串列生產出 x。

兩個生產子會形成卡式積，
最右的生產子頻率最快，實例如下：

xs = [1,2]
ys = [3,4]

[(x,y) | x <- xs, y <- ys]

產出結果如右：

[(1,3),(1,4),(2,3),(2,4)]


:條件式:

生產子後放布林式，則能形成條件式，用來過濾結果串列之值。

如快排可定義如下：

quicksort []     = []
quicksort (x:xs) =  quicksort [y | y <- xs, y < x]
                 ++ [x]
                 ++ quicksort [y | y <- xs, y >=x]
